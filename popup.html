Keybinds:

Select (Keyboard): Hold Alt + s and click on an event. Or, drag mouse to select multiple

Select (Mouse): Hold Middle Mouse Button + LeftShift and click on an event. Or, drag mouse to select multiple

Unselect All Events: Alt + a

Move Selected Events: Alt + b

Delete Selected Events: Alt + Delete OR Alt + Backspace
